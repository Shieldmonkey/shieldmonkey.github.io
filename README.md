# Shieldmonkey (Website)

This repository hosts the official website for **Shieldmonkey**, a security-first UserScript manager.

The website content is located in the `/docs` directory and is served via GitHub Pages.

## Official Website
https://shieldmonkey.github.io/

## Main Repository
- [Shieldmonkey Extension](https://github.com/shieldmonkey/shieldmonkey) - The source code for the browser extension itself.
