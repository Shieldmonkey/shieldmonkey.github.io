Shieldmonkey

Security-First UserScript Manager

Shieldmonkeyは、徹底したサプライチェーン対策と透明性の高いオープンソース開発により、セキュリティを維持しつつもユーザースクリプトによる利便性をもたらす拡張機能です。

The "Static" Philosophy

不便さと引き換えに手に入れる、絶対的な安心。
Shieldmonkeyは「何もしない」ことであなたを守ります。

Zero Network Activity

拡張機能自体が外部との通信を一切行わないため、スクリプトの自動更新やクラウドバックアップも存在しません。この制約はCSP (connect-src 'self') によって物理的に担保されています。

No Silent Updates No Cloud Backup

Supply Chain Defense

NPMのベストプラクティスを徹底。npm ci による完全なバージョン固定と、ignore-scripts による実行防止で、ビルド環境の安全性を確保します。

Verified Dependencies

Verifiable Trust

セキュリティツールがブラックボックスであってはなりません。ビルド後のコードに対しても意図的にMinify（圧縮）を行わず、Source Mapsも同梱することで、誰もが検証可能な透明性を提供します。
